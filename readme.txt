How to use:
Make sure python is installed
If running for first time run this command in command prompt:

pip install -r requirements.txt

This wil install requried packages for it to function.  


Place main.py inside your target folder which should be setup in the following format:
formgroup1
	image1
	image2
	image3
formgroup2
	image1
	image2
	image3
formgroup3
	image1
	image2
	image3

The output will be in the same format inside the newly created cropped folder.
